# movie_api

### Description

Movie API for managing movie database

### Endpoints

/movies
/movies/movie/[name]
/movies/movie/genre/[name]
/movies/movie/director/[name]
/user
/users/[username]
/user/username/favmovies/[Movie]
/user/username/favmovies/[MovieID]
/user/[nameID]
