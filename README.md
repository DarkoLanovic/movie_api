# movie_api

### Description

Movie API for managing movie database

### Endpoints

/movies
/movies/movie/[name]
/movies/movie/genre/[name]
/movies/movie/director/[name]
/user
/users/[username]
/user/username/favmovies/[Movie]
/user/username/favmovies/[MovieID]
/user/[nameID]

### Dependencies

- "body-parser": "^1.19.0",
- "express": "^4.17.1",
- "lodash": "^4.17.21",
- "morgan": "^1.10.0"

### Tecnologies:

- Node
- Express

**To run app**
`npm start`
