A JASON object holding data about all the movies
Show data about a single movie by title /movies/title/:title GET _ A JASON object holding data about a single movie,
contains movie ID, name, year, genre,
director, description and image
{
title: "African Queen",
actors: "...",
image: "...",
genre : "...",

}
Show movies by genre /movies/genre/:genre GET _ A JASON object holding data about movie genre
Show data about a director by name /movies/director/:director GET _ A JASON object holding data about a director by name,
contain bio, birth year, death year
